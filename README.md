> Banking made simple. Bankist is the modern way to manage your money — smarter, faster, and completely online.

Bankist is a minimalist, **desktop-only** marketing website for a fictional digital bank. This is a pure frontend project — no backend, no database, no framework. It was built as a **DOM manipulation practice project**, exploring a range of JavaScript browser APIs and UI interaction patterns.

> ⚠️ **Not responsive.** This site is designed for desktop viewports only.

### What's Practiced

- **Intersection Observer API** — section reveal animations on scroll, sticky navigation
- **Tabbed component** — switching between feature panels
- **Slider / carousel** — navigating slides with buttons and keyboard
- **Lazy loading images** — images load as they enter the viewport
- **Smooth scrolling** — nav links scroll to sections smoothly
- **Link hover effects** — sibling links fade on hover using event delegation
- **Modal window** — open/close sign-up prompt

### Tech Stack

- **HTML5** — semantic markup
- **CSS3** — custom properties, flexbox, grid
- **JavaScript (ES6+)** — DOM manipulation, events, Intersection Observer API

### Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/bankist-website.git
   ```

2. Navigate into the project folder:

   ```bash
   cd bankist-website
   ```

3. Open `index.html` in your browser — no build tools or dependencies required.
